# Le Flavor Network : Cartographie des Saveurs 

Bienvenue sur le dépôt officiel du projet **Flavor Network**, réalisé dans le cadre de l'enseignement "Graphes et OpenData" (Licence MIAGE).

## Objectif du Projet
Ce projet explore la théorie scientifique du **Food Pairing** à l'aide de la théorie des graphes. L'objectif est de prouver mathématiquement que certains aliments s'accordent en cuisine non pas par hasard, mais parce qu'ils partagent un profil chimique commun (molécules d'arôme).

## Démonstration en ligne
